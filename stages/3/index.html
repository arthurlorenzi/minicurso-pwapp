PWA Docs

PWA Docs é uma aplicação desenvolvida para o minicurso "Introdução a Progressive Web Apps" ministrado na Semana da Computação 2017 do DCC-UFJF. A aplicação permite a criação e edição de documentos de texto de maneira colaborativa entre os usuários, verificando incompatibilidades entre versões de um documento editado por múltiplos usuários. Todo o server-side foi implementado em NodeJS.
